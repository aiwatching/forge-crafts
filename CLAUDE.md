# forge-crafts

Public registry for [Forge](https://github.com/aiwatching/forge) Crafts. This
repo only hosts metadata + craft source files; nothing here runs.

Contributions arrive as PRs from Forge's publish flow — see `README.md` for
the format. Merging is handled by the maintainer (manually or via their own
Forge automation). No special setup needed in this repo.
